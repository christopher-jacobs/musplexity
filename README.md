# Musiplexity: Classifying Music by Genre Using Data Analysis and Machine Learning

This is project I conducted as part of an internship with the Patriot Machine Learning Research Group at Francis Marion University. This research analyzes how well machines can sort musical selections into predetermined genres using persistent homology and k-means clustering.

Persistent homology is a subsection within topological data analysis that studies features within graphical planes. These features or "holes" give a unique identity and allow for more broad categorizations when handling larger scale datasets. This project utilizes the Vietoris-Rips complex, a way to form a topological space by measuring the distance from a given set of points.
